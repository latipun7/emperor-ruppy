## Getting Started

Getting up and running is as easy.

1. Make sure you have [NodeJS](https://nodejs.org/) and `yarn` installed.
2. Install your dependencies

   ```bash
   cd path/to/emperor-ruppy
   yarn
   ```

3. Start dev app

   ```bash
   yarn dev
   ```
